This is a Basic Inventory Management System .
Provides a real-time web-based dashboard for managing inventory, orders, suppliers, warehouses, and shipments.
Enables CRUD operations (Create, Read, Update, Delete) on key supply chain data via interactive forms and modals.
Uses a structured relational database (SQLite) to store critical data reliably and consistently.
Designed with future offline-first architecture and AI + IoT integration in mind.
Provides real-time data visualization (KPIs, charts) based on database content
